# Icons

Place your app icon files here. The image you provided should be converted to the following formats and sizes:

Required icon files:
- `32x32.png` - 32x32 pixels
- `128x128.png` - 128x128 pixels
- `128x128@2x.png` - 256x256 pixels (retina)
- `icon.icns` - macOS icon format
- `icon.ico` - Windows icon format

You can use online tools or image editing software to convert your image to these formats:
- PNG files: Use any image editor (GIMP, Photoshop, etc.) to resize
- ICO file: Use tools like https://convertio.co/png-ico/ or ImageMagick
- ICNS file: Use tools like https://cloudconvert.com/png-to-icns or iconutil on macOS

Place the converted files in this directory, then rebuild the Tauri app.
