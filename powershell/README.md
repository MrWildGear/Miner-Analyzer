# PowerShell Scripts - EVE Miner Analyzer

## Available Scripts

- **rolled Ore Strip miner.ps1** - Analyzer for ORE Strip Miner
- **roll_analyer_modulated.ps1** - Analyzer for Modulated Strip Miner II
- **roll_analyzer_modulated_realworld.ps1** - Real-world performance analyzer for Modulated Strip Miner
- **mutaplasmid_odds_calculator.ps1** - Mutaplasmid odds calculator

## Usage

Run any script directly:
```powershell
.\rolled Ore Strip miner.ps1
```

## Features

- Real-time clipboard monitoring
- Automatic tier assignment
- Mutation percentage calculations
- Color-coded output
